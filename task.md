# 目的
我要舉辦一場比賽，因此需要建立一個可以讓所有人上來填寫的報名網站。

# 表單規格
參賽者必需填寫：
1. 姓名(text)
2. 選擇部門 (select: 電子商務部/流通事業部/永續發展部/保險科技部/數據加值部/事業發展部)
3. 員編號碼(text)
4. 想要的時段(radio input)： 抓取"google sheet"裡的 "時段選項"sheet裡的"時段"欄位下的所有資料，為選項
5. 想選擇的角色(radio input): PM/SA/PG/業務/客服/行政

## 開發規格
- 樣式使用scss
- 最後將發布於github page
- 我的開發經驗不多，請為我選擇好上手的開發框架

### 資料庫
串google sheet，包含以下sheet tab及其欄位：
- 表單資料送出後，所有資料儲存於"google sheet A"上
- 全體職員：姓名、部門、員編
- 時段選項：時段、已選人數、狀態(可選/已額滿)
- 結果：送出時間、姓名、員編、選擇時段、角色

## 前端規則
- 想要的時段，必須於畫面上顯示目前已有多少人選擇，額滿必須把選項disable
- 所有題目皆為必填，尚未填寫需有警示
- 送出失敗時，需顯示警示談彈窗說明
  1. 您的資料填寫錯誤，請再次確認姓名、員編、部門是否正確
  2. 很抱歉，您選擇的時段已額滿，請重新選擇 

## 驗證規則
- 參賽者填寫的 姓名 與 員編號碼，必須與"google sheet" 的"全體職員" 裡的"姓名"、"員編"、"部門"進行核對，必需(1)確實有這個姓名， (2)且員編正確填寫，(3)且部門正確 才能夠送出。
- 每個時段最多8個人，已額滿不能送出

## 網站風格
- 現代化、可愛風、顏色繽紛
- radio input 請以方塊按鈕型、邊角 radius 4px、點擊active明顯的樣式呈現
- 想要的時段會有很多個，請先預設有10個，資料你先自行產出，後面再去抓取google sheet
